<h1> Bus_Seat_Management_Backend </h1>

<h2> For running server </h2>

1. **Run server**
```bash
    uvicorn main:app --reload
    ```


<h2> For Installation Purpose </h2>

1. **Fast API**
    ```bash
    pip install fastapi
    ```

2. **Uvicorn**
    ```bash
    pip install "uvicorn[standard]"
    ```
    
3. **Aiosqlite**
    ```bash
    pip install "aiosqlite"
    ```
